## Descripcion e Informacion Importante
Este programa ha sido desarrollado en **Python** aplicando la **arquitectura por capas** y la conexión a **SQL Server** vistas en clase.  

Para lograr la conexión entre Python y SQL Server se ha usado la dependencia **pyodbc**, la cual se obtiene ejecutando desde la consola el siguiente comando:

```bash
pip install pyodbc

```
## Base de Datos

La base de datos desarrollada en SQL Server se encuentra en formato de BackUp dentro de este repositorio con el nombre:
```text
backupUsuarios.bak
```

## Ejecución del Programa

Para ejecutar el programa se recomienda usar la consola **CMD**.

1. Abrir la consola en la carpeta del proyecto.
2. Ejecutar el siguiente comando:
```bash
python main.py
```
